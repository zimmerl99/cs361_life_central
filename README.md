**--Life Central--**<br>
This program is a planner and task tracking app that allows you to organize your life and maximize efficiency day to day.

**--Important Notes--**<br>
This program requires full usage of the database api(required for usage), the inspirational quotes api(not required for usage), and the sort by date api (not required for usage). Links to those resources can be found here:
<br>https://github.com/zimmerl99/database_microservice361
<br>https://github.com/zimmerl99/inspirational_quotes
<br>https://github.com/zimmerl99/sort-by-date

<br> Also you must download these dependencies: pip install PyQt6, pip install requests, pip install fastapi uvicorn (for the api's)
